Ի սկզբանե, GitHub.com-ում գրանցվեցի, մի քիչ ուսումնասիրեցի Git-ն ու GitHub-ը, հասկացա, որ լրիվ տարբեր բաներ են։ Հետո փորձեցի հասկանալ խնդրի պահանջը․ հիշելով, որ http-ն ձևավորվում է request և response միջոցով, հասկացա, որ պիտի այնպիսի ծրագիր գրել, որ localhost-ով զննարկիչում request ուղարկելիս response-ը լինի "Hello world"։ 
Ծրագիրը js-ով պիտի գրեինք, որի syntax-ին լավ ծանոթ չեմ, հետևաբար՝ մտա Youtube, փնտրեցի "Как создать веб-сервер с js", միանգամից բերեց Node.js-ի մասնակցությամբ տեսանյութեր։ Քաշեցի node.js, սկսեցի կոդերը ուսումնասիրել ու պարզվեց, որ հենց սերվեր ստեղծելու հրաման գոյություն ունի ու դրա հետ մեկտեղ req, res արգումենտներ կարող ենք տալ։
Visual Studio Code-ում եմ փորձել ծրագիրը։ Հերթը հասավ ծրագիրը GitHub ներբեռնելու։ Բայց սկզբից պիտի Git ծրագիրը ներբեռնեի ու միացնեի GitHub-ի account-ին։ Ընկերներիս օգնությամբ cmd-ով հրամաններ գրեցինք, որը միանգամից բացել էի այն պանակի մեջ, որտեղ գրված ծրագիրն էր։ Արդյունքում ստացվեց․․․
